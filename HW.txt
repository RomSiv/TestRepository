 Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
Командой git clone путь репозитория
Предварительно должен быть добавлен публичный ключ ssh в настройки GitHub


Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

Создаём папку в нужной нам директории в которой будем оставлять нужные нам дня загрузки на GitHub файлы.
Командой git init создаём в папке файл .git, который будет отвечать за все перемещения между ПК и GitHub
После создаём новый репозиторий на GitHub (что бы не путаться называем его как наша папка на ПК) без файла readme.
После нажатия кнопки создать на странице покажутся команды, которые нам нужно доделать, а именно:
git remote add origin .... - сращиваем наш репозиторий с нашей папкой
git branch -M main - меняем мастер на мэйн
git push -u origin main - делаем первый пуш именно с ключом -u  


Как получить на локальный компьютер изменения с github.com
Командой git fetch

Какую информацию нам дают команда :
git status  - статус добаленых файлов 
git log - история добавленных файлов в репозиторий от старых к новым
git remote -v - показывает привязана ли папка на локальном компьютере к GitHub
